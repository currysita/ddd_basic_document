# このリポジトリについて

## 目的

書籍「実践ドメイン駆動設計から学ぶDDD実装入門」を読んで学習するための個人用学習ドキュメント集。
章ごとのまとめ・用語集・問題集を作成し、繰り返し学習（複数周）に活用する。

## 問題集の運用方針

- 問題集はカテゴリ（A〜Z）単位で管理し、各カテゴリは複数章にまたがることがある
- 問題IDは `カテゴリ名-連番`（例：`D1-03`）
- 学習は「周」単位で管理し、進捗ファイルで状態（未着手／学習中／完了）を追跡する
- AIレビューの結果はレビュー記録ファイルに残す（将来の比較・改善のため）

# Claude Code 作業上の注意事項

## ファイル検索

- ファイルの存在確認をする際、拡張子を決め打ちしないこと。
  - 例：`*.jpg` で検索して「ファイルがない」と判断したが、実際は `.png` だった。
  - **常に `*` パターンで検索し、実際の拡張子を確認してから次の操作を行う。**
